Comment fonctionnent les parrainages ?

1

Choisissez une offre

Sélectionnez le programme de parrainage qui correspond à vos besoins financiers.

2

Utilisez le lien ou code

Cliquez sur le lien de parrainage ou utilisez le code lors de votre inscription.

3

Créez votre compte

Suivez les instructions pour finaliser la création de votre compte.

4

Recevez votre récompense

Après validation de votre compte, recevez votre bonus de parrainage.
